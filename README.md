## Getting Started
To get a local copy up and running follow these simple steps.
### Prerequisites

Have ```node``` installed.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Didiloy/VBS.git
   ```
2. Navigate to the directory named vbs
     ```sh
    cd VBS/vbs
     ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Launch the developpement server
   ```sh
   npm run serve
   ```
